# CindyFlow - Modern Startup Landing Page

> A modern, high-conversion, and fully responsive single-page landing page designed for a fictional AI-driven productivity suite.

## Overview

CindyFlow is a state-of-the-art landing page built to demonstrate advanced frontend development techniques without the use of heavy frameworks. The design focuses on a rich dark-theme aesthetic, utilizing glassmorphism, dynamic grids, and smooth scroll animations to create a premium user experience.

## Features

- **Framework-less Build**: Crafted entirely with semantic HTML5, modern CSS3, and Vanilla JavaScript.
- **Responsive Layout**: Fluid design built on CSS Grid and Flexbox, scaling perfectly from desktop down to mobile.
- **Scroll Animations**: Dynamic reveal animations powered by the `IntersectionObserver` API.
- **Complex Layouts**: Features an asymmetrical grid system and hover-depth interactive pricing cards.
- **Glassmorphism UI**: Beautiful, translucent sticky navigation and UI components with background blur.

## Technology Stack

- **HTML5**: Semantic document structure.
- **CSS3**: Variables, CSS Grid, Flexbox, media queries, and keyframe animations.
- **JavaScript (ES6+)**: DOM manipulation and Intersection Observer for triggering animations.

## Running Locally

1. Clone this repository:

   ```bash
   git clone https://github.com/cindydike/modern_startup_landing_page.git
   ```

2. Navigate into the directory:

   ```bash
   cd modern_startup_landing_page
   ```

3. Open `index.html` directly in your browser, or start a local development server:

   ```bash
   # If you have Python installed:
   python -m http.server 8000
   ```

4. Visit `http://localhost:8000` in your browser.
